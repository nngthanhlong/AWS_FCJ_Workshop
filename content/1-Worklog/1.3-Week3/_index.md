### Week 3 Achievements:

* Understood the overall AWS Compute services and how to use related components:
  * Amazon EC2
  * Auto Scaling
  * Amazon Lightsail
  * EFS, FSx
  * AMI, Snapshot

* Mastered various EC2 instance types and specifications:
  * General Purpose, Compute Optimized, Memory Optimized, Storage Optimized, GPU
  * vCPU, RAM, Burst, Network Performance parameters

* Proficient in EC2 operations:
  * Launch and manage Windows and Linux EC2 instances
  * RDP/SSH connection and install basic packages
  * Manage Security Groups, open necessary ports for applications
  * Practice start, stop, reboot, terminate, and check instance status

* Understood and deployed AMI, snapshot, and backup:
  * Create custom AMI from EC2
  * Practice EBS snapshot
  * Restore EC2 from AMI/snapshot

* Practiced basic Auto Scaling:
  * Create Launch Template
  * Create Auto Scaling Group
  * Verify automatic scale out/scale in capability

* Deployed applications on EC2:
  * Deploy User Management Application on Amazon Linux 2
  * Deploy Node.js Application on Windows EC2
  * Verify connectivity from client and web browser, ensure stable operation

* Managed costs and access permissions:
  * Configure IAM, roles, policies
  * Monitor usage reports & budget
  * Identify potential costs and optimization methods

* Cleaned up resources after hands-on practice:
  * Terminate EC2, delete volumes, detach AMI, delete Security Groups
  * Keep environment clean for next week

* Comprehensive skills:
  * Understand relationship between EC2, EBS, AMI, Auto Scaling
  * Able to deploy, test, and manage Compute environment on AWS independently
