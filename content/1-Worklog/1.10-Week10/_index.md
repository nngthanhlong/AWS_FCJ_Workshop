### Week 10 Achievements:

* Understood how to use AWS Cloud9 IDE:
  * Main features: code editor, terminal, debugger, support multiple programming languages.
  * Know how to customize interface based on preference (color theme, keybinding, syntax highlight).

* Practiced creating and managing Cloud9 environment:
  * Create new Cloud9 environment, choose instance type, configure access permissions.
  * Check AWS connection and use integrated terminal.

* Proficient in basic operations on Cloud9:
  * Create, edit, save and delete files/folders.
  * Use terminal to run basic commands and check environment.

* Used AWS CLI in Cloud9:
  * Check account info, region, service list.
  * Practice basic operations with EC2, S3 via CLI.

* Managed and cleaned up resources:
  * Delete Cloud9 environment, terminate instance.
  * Document process and notes when using Cloud9 for subsequent labs.

* Comprehensive skills:
  * Confident using Cloud9 to write, run and manage source code on AWS.
  * Combine CLI and IDE usage to deploy labs efficiently.
